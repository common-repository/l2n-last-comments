== Description ==
WordPress comes with a recent comments widjet, wich displays the name of the commentator comments with a link to the comment and the title of the post.
l2n last comments, displays the name of commentator with an excerpt of the comment and a link to it, which gives a better idea about the comment. 
Note : this widjet is in Arabic language.
Enjoy it ...

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a Widjet : l2n last comments.
